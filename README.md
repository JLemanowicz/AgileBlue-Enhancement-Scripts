# AgileBlue-Enhancement-Scripts
TamperMonkey Scripts for Automation

## Required
<a href="https://chromewebstore.google.com/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo?utm_source=item-share-cb">TamperMonkey Browser Extension</a>

*To install a script, click an Install Link after the TamperMonkey Extension is installed.*

| Context Menu | Highlight and right-click any IP, SHA256, or Elastic Log ID to open an applicable link for analysis | https://github.com/JLemanowicz/AgileBlue-Enhancement-Scripts/raw/refs/heads/main/Scripts/Context-Menu.user.js |
